Бесплатный и простой телеграмм бот для группового чата. 

Вы можете установить его в свою группу, авторским правом он не защищен.


___О Боте V1__

Бот простейшего уровня. 
Из возможностей:
1)Играть в игры. В качестве игр выбраны смайлы с анимацией и функцией рандома (по типу боулинга, мячика и т.п.)
2) Откликается на слова. Команды не используются

__Как запустить____
1)В файл бота вставьте api из botfather
2)Поставьте файл на хостинг/сервер и запустите его.
